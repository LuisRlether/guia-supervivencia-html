¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Según Mozilla Developer Network (MDN), HTML permite definir textos, imágenes, enlaces y otros elementos que el navegador interpreta para mostrar una página web.

Sintaxis básica

La sintaxis de HTML se basa en el uso de etiquetas escritas entre los símbolos menor que (<) y mayor que (>). La mayoría de las etiquetas cuentan con una etiqueta de apertura y una de cierre.

Ejemplo de estilo inline aplicado directamente en una etiqueta HTML. 

<!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

Etiquetas HTML más utilizadas

HTML cuenta con una gran variedad de etiquetas. A continuación se presentan algunas de las más utilizadas según la documentación de MDN.

  • <h1> a <h6> – Definen títulos
  • <p> – Define párrafos
  • <img> – Inserta imágenes
  • <a> – Crea enlaces
